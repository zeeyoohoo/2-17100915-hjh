My Favorite Beer

가만히 있어도 더운 여름. 이럴 때일 수록 시원한 맥주 한 캔이 그렇게 생각난다. 친구들과 얘기를 하던, 집에서 혼자 영화를 보던, 즐거운 순간에 맥주는 나와 항상 함께했다. 길다면 길고 짧다면 짧은 3년 동안 내가 마셨던 맥주들 중 가장 좋아하는 맥주에 대해 매우 개인적이고 주관적인 이야기와 함께 소개해보려 한다.

제주 위트 에일

COUNTRY : 대한민국 제주도

TYPE : Ale

ABV : 5.3%

IBU : 16

SCENT : 감귤피

STORY : 특별한 맛은 아니지만 에일이 생각날 때 편하게 먹기 좋다. 산뜻하고 마실수록 맛이 진해지는 느낌이다. 처음 나왔을 땐 제주도에서만 판매해서 너무 먹어보고 싶었는데 요즘은 편의점에도 들어오더라. 그 덕에 자주 마시게 되었다.

칭따오

COUNTRY : 중국

TYPE : Pale Lager

ABV : 4.7%

IBU : 25

SCENT : 밀

STORY : 깔끔하고 드라이해서 기름진 안주와 잘 어울린다. 괜히 양꼬치엔 칭따오라는 말이 존재하는 것이 아니다. 실제 칭다오 사람들은 양꼬치보다 바지락을 최고의 안주로 생각한다고 한다는데, 적어도 나한테 칭따오+양꼬치 조합은 공식이다.

오비라거

COUNTRY : 한국

TYPE : Pale Lager

ABV : 4.7%

IBU : 2.71

SCENT : 밀

STORY : 베이직한 맥주가 생각날 때 마시기 좋다. 한동안 에일에 질려서 마실 맥주가 필요했을 때 자주 마셨는데, 친구들과 가볍게 맥주 한 잔 할 때 편하게 마시기 좋다.

크로넨버그 1664 블랑

COUNTRY : 프랑스

TYPE : Ale

ABV : 5.3%

IBU : 16

SCENT : 과일

STORY : 향긋한 과일향과 부드러운 청량감을 가지고 있는 프랑스 국민맥주다. 당연한 말일지도 모르지만 생맥이 더 맛있다. 길가다 음식점에서 이 맥주의 생맥 간판을 보면 눈여겨 봐두고 언젠가 꼭 한번씩 찾아간다.

>

카스

COUNTRY : 한국

TYPE : Pale Lager

ABV : 4.7%

IBU : 2.65

SCENT : 밀

STORY : 크로넨버그가 프랑스의 국민맥주라면 한국의 국민 맥주는 카스다. 내가 마신 첫 맥주이기도 하다. 소맥으로 먹기 제일 잘 어울린다고 생각한다. 그냥 먹기엔 쇠맛이 나는 기분. 그렇지만 축구 경기를 틀고 새우깡을 꺼낸다면? 답은 하나, 무조건 카스를 마셔야 한다.

>

블루문

COUNTRY : 미국

TYPE : Wheat Ale

ABV : 5.4%

IBU : 11

SCENT : 오렌지, 고수

STORY : 거의 최애 맥주라고 할 수 있다. 오렌지향, 고수향을 가지고 있다. 하나 꿀팁을 주자면, 오렌지 가니쉬를 더하면 훨씬 맛있다.

>

써머스비

COUNTRY : 덴마크

TYPE : Apple Cider

ABV : 4.5%

IBU : -

SCENT : 사과

STORY : 사과향으로, 유사한 맥주로는 매그너스, 애플폭스가 있다. 평소 한강에서 친구들이랑 자주 마셔서 이 맥주를 보면 한강이 생각난다. 사과맛이라서 가볍게 마시기 좋지만 도수가 낮지는 않다. 마시고 나서도 사과맛이 남아있어 기분이 좋다.
